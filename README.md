# 📺 FireTV 多端影视播放器

一款适配 **Android & iOS 移动端** 和 **Windows & Mac PC端** 的影视播放应用。
灵感来源于 **MoonTV**，在此基础上针对 **资源管理、播放体验和可用性** 做了大量增强和优化。

---

## ✨ 项目简介

本项目是一款基于 **MACCMS JSON API** 的多端影视播放器，支持海量影视资源聚合、智能资源优选、贴片广告去除等功能，致力于提供 **更干净、更稳定、更流畅** 的观影体验。

- 📱 同时支持 Android / iOS / Windows / Mac
- 🔌 兼容标准 MACCMS JSON API 接口
- 🚀 自动测速 & 资源优选播放

---

## 🚀 核心功能

### 1️⃣ 首页视频榜单

- 热门影视推荐
- 最新更新榜单
- 分类聚合展示
![home](https://github.com/user-attachments/assets/c3454f38-5fe5-4a98-af4a-fd6f9c50d95e)

---

### 2️⃣ 看剧页 · 视频筛选

- 按 **类型 / 地区 / 年份 / 排序** 等维度筛选
- 快速定位目标影视内容
![movie](https://github.com/user-attachments/assets/c26c01a8-dba5-4bcd-8102-5b705afa9bd5)

---

### 3️⃣ 视频搜索

- 支持关键词搜索
- 多资源站统一搜索结果
- 搜索结果自动合并与去重

---

### 4️⃣ MACCMS JSON API 资源支持

- 完整支持 **MACCMS 标准 JSON API**
- 内置大量优质影视资源站
- 支持以下资源管理方式：
  - ➕ 单个资源站手动添加
  - 📦 通过订阅地址批量订阅资源站
- 可自由启用 / 禁用指定资源站
![resources](https://github.com/user-attachments/assets/777e1d2d-cc93-4b85-ba41-7e1ea4025d4a)

---

### 5️⃣ 去除贴片广告

- 自动识别并跳过视频中的 **贴片广告**
- 减少播放中断，提升观影体验

> ⚠️ 说明：该功能基于播放链接分析与资源筛选实现，不对视频内容本身进行修改。
![setting](https://github.com/user-attachments/assets/a06921f2-efc7-4132-bcec-fc9dfdc04054)

---

### 6️⃣ 测速 & 视频优选播放

- 播放前对多个视频源进行自动测速
- 探测内容包括：
  - 是否可正常访问
  - 实际加载速度
- 自动选择 **可达性最佳、速度最快** 的视频源进行播放
- 有效避免卡顿、失效链接和长时间加载问题
